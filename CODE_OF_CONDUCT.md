# 📜 贡献者行为准则（Contributor Covenant v2.1）

## 我们的承诺

为了营造开放、友好的社区环境，我们作为贡献者与维护者，承诺让每个人都能参与本项目，
无论年龄、体型、可见或不可见残障、族群、性别、性取向、经验水平、教育背景、社会经济地位、国籍、相貌、宗教或性取向。

我们承诺以有助于构建开放、友善、多元、健康社区的方式行事与互动。

## 我们的准则

有助于营造积极环境的行为包括：

- 以同理心与善意对待他人
- 尊重不同的观点、经验与立场
- 真诚、建设性地给出与接受反馈
- 对社区整体最有利的事负责
- 对其他社区成员展现包容

不可接受的行为包括：

- 使用性暗示的语言或图像，以及任何性质的骚扰
- 挑衅、侮辱性或贬低性评论，人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，公开他人的私人信息（如住址、邮箱）
- 其他在职业场合中可能被合理认定为不当的行为

## 我们的责任

维护者负责界定可接受行为的边界，并对外显违规采取公平、适当的纠正措施。
维护者有权且应当将不符合本准则的评论、提交、代码、Issue 与 PR 移除、编辑或拒绝。

## 适用范围

本准则适用于所有项目空间，以及当个人在公开场合代表项目时的对外沟通
（例如使用官方邮箱、社交媒体账号，或在会议中担任项目代表）。

## 执行

可向项目维护者报告骚扰或其他不可接受行为。所有投诉都会被及时、公正地审查与处理。
维护者有义务保护举报者的隐私。

## 归因

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，版本 2.1，
其文本可从 https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/ 获取。
